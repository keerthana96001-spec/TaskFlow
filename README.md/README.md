#  TaskFlow

TaskFlow is a modern and responsive To-Do List web application built using HTML, CSS, and JavaScript.

##  Features

- ✅ Add Tasks
- ✅ Delete Tasks
- ✅ Mark Tasks as Completed
- ✅ Press Enter to Add Tasks
- ✅ Local Storage Support
- ✅ Task Counter
- ✅ Responsive Design

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
